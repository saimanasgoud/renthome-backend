# 🏠 RentHome Backend

A full-stack backend application for a property rental platform built using Spring Boot. This project provides RESTful APIs for managing users, properties, analytics, and authentication.

## 🚀 Features

- 🔐 User Authentication (JWT & OTP/Magic Link)
- 🏡 Property Management (Add, View, Manage Listings)
- 🖼️ Media Upload (Cloudinary Integration)
- 📊 Analytics Tracking
- 📧 Email Services (OTP & Notifications)
- 🗄️ MySQL Database Integration

## 🛠️ Tech Stack

- Java 17
- Spring Boot
- Spring Data JPA (Hibernate)
- MySQL
- Cloudinary
- Maven
